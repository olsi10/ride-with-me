
# Django 네트워크 프로그래밍 수행평가

## 국토종주 - A Whole New World!

**간단 서비스 소개**
- 자신만의 자전거 트랙을 공유하는 게시판 형식의 웹사이트


**서비스 기획 배경**
- 자전거 타는 것을 좋아하기도 하고 자전거를 타는 트랙은 항상 같았기 때문에 추천받는다면 좋을 것 같다는 생각에 기획함

**서비스 효과**
- 매번 같은 자전거 트랙을 타지 않아도 됨
- 다른 사람의 트랙을 참고해서 자신만의 트랙을 만들 수 있음
